Description of the columns

  • Trait group: trait categories used in the Global Biobank Engine. Its FAQ page has more information.
  • Trait: a trait in UK Biobank.
  • Family: it represents whether the PRS model is fit with gaussian model (for continuous response, such as standing height) or binomial model (for binary outcomes, such as disease). We use R2 and AUC as the evaluation metric for gaussian and binomial models, respectively. In this table, we display those evaluation metric computed on the hold-out test set consits of individuals of whiter British ancestry in UK Biobank cohort.
  • Geno: the predictive performence (R2 or AUC) of the predictive model with genotype features alone. Note that this model does not include genotype PCs.
  • Covar: the predictive performence (R2 or AUC) of the predictive model with covariates (age, sex, and genotype PCs) alone.
  • Full: the predictive performence (R2 or AUC) of the full model that have both genotypes and covariates alone.
  • delta: the incremental predictive performence (R2 or AUC) of the full model against the covariate-only model, i.e. the difference between the values in "Full" and "Covar" columns.
  • # variants: the number of genetic variants included in the model
  • P (WB): P-value of the incremental predictive performance of the PRS model evaluated on the test set individuals of white British ancestry in UK Biobank. The p-value is before the multiple hypothesis correction.
  • significant?: this indicator variable shows whether the PRS model has a significant (P < 0.05/2000) incremental predictive performance.

Version info

  • 2021.11.04. Version 2.0.0: We made several changes. Of note, the coefficients of the PRS models does not change for most traits with the exception of the biomarker traits (please read (3) below). (1) We found a bug in our performance evaluation script, which resulted in inaccurate statistical significance reported in the previous version. After fixing the issue, we saw an increase in significance for many PRS models, resulting in total of 813 significant (p < 2.5e-5) PRS models. (2) Starting this version, we have also included an indicator variable denoting the types of genotyping arrays in covariates when evaluating the predictive performance. (3) For the blood and urine biomarker traits analyzed in our previous study (Sinnott-Armstrong*, Tanigawa*, et al 2021), we used two versions of traits (one from the raw measurement values and the other after the covariate adjustment). Starting this version, we have removed those duplicates. Specifically, we took the PRS models from our biomarker paper, which we computed for the covariate-adjusted traits, and evaluated their performance for the raw phenotypes, so that the incremental predictive performance has meaningful values.
  • 2021.09.23. Version 1.3.1: We improved the clarity of the table and Website. We added a link to our manuscript.
  • 2021.08.23. Version 1.3: We add plots that compares the number of active variables in the PRS model vs the incremental predictive performance of the PRS model.
  • 2021.07.23. Version 1.2: We add the p-value of the PRS model as Well as an indicator variable of the significance of PRS. We also removed some of the duplicated traits.
  • 2021.01.16. Version 1.1: Initial public release of the results
